A tribute page is a simple static website

  1. You are required to have knowledge in HTML and CSS
  2. Open your working directory
  3. Create a new folder eg "Tribute Page"
  4. Open your text editor of choice
  5. Create a new file index.html"
    • Save it to your working directory
    • Write the openning and closing html tags and start filling in your content
  6. Create another new file "style.css"
    • Save it to your working directory
    • Use this to style your website
  7. Open your folder directory
  8. Double-click the index.html file and let it open in your browser of choice.
  9. Continuing styling while refreshing the page to update your new style.
